# 体癖チェッカー 判定項目リライト案

## 改善の目的

現行の項目は元の特徴づけには忠実だが、自己申告チェック式で使うと判定の中立性を損なう偏りがある。本リライトは以下4点を是正する。

1. **社会的望ましさバイアスの除去** — 3・5・10種に肯定的項目、7・8・9種に否定的項目が偏在し、真の体癖に関わらず前者が水増し・後者が過小評価される。全種を「価値中立な観察文」に揃える。
2. **身体・観察項目の底上げ** — 体癖は本来「体の癖（姿勢・重心・体型）」が一次情報。各種2〜3個確保し、心理項目偏重による判定の不安定さを抑える。
3. **動作・顔項目の追加** — 「無意識の動き」「顔の骨格」は望ましさバイアスが乗りにくく差別化力が高い。各種1個ずつ追加する。
4. **ダブルバーレル・重複・誤字の修正** — 1項目1主張に統一し、種をまたぐ重複表現を差別化する。

各種10項目は維持（既存の集計ロジック `counts` をそのまま使えるため）。
タグ凡例：【身】=体型・姿勢、【顔】=顔の骨格・特徴、【動】=無意識の動き・癖、【心】=心理・行動

## 集計設計への補足提案（任意）

タグを `cb.dataset.kind` に持たせることで、種スコアの内訳を見られるようになる。

```
body   = 【身】【顔】【動】
mind   = 【心】
```

- 【動】【顔】は望ましさバイアスが低いため、`kind=body` に重み（例：×1.5）を付けると判定精度が上がる
- 【身】【心】を別集計し、レーダーチャートを2枚出して整合を見る方法も有効

5段階評価（-2〜+2）への移行時は、URLパラメータのセパレータを `-` から `,` に変更し、表示時に `(sum + 20) / 40 * 100` で0〜100に再スケールすること。

## 集計ロジックへの制約ルール追加（要実装）

体癖理論には「ありえない組み合わせ」という構造的制約がある。結果表示時にこれを反映する。

### 制約1：同一軸の陽陰は同時に高スコアにならない

各軸の陽陰ペアは排他的。両方が高スコアで拮抗している場合は、判定が不安定であることを示す。

| 軸 | 陽 | 陰 |
|---|---|---|
| 上下 | 1種 | 2種 |
| 左右 | 3種 | 4種 |
| 前後 | 5種 | 6種 |
| ねじれ | 7種 | 8種 |
| 開閉 | 9種 | 10種 |

**実装案**：上位結果に同一軸のペアが両方含まれる場合、「1種と2種が拮抗しています。どちらか一方を選ぶため、身体項目の回答を見直してみてください」のような注意書きを表示する。

### 制約2：全て奇数種・全て偶数種はありえない

体癖は複数種の複合で成り立つが、奇数種（陽）だけ、または偶数種（陰）だけが並ぶ組み合わせは理論上存在しない。必ず陽陰が混在する。

**実装案**：上位3〜5種が全て奇数または全て偶数だった場合、「回答に偏りがある可能性があります。陽（奇数種）と陰（偶数種）が混在するのが自然な結果です」と表示する。

### 判定の信頼度スコア（任意）

上記2制約への違反数と、同一軸の拮抗度合いをもとに「この結果の信頼度：高／中／低」を表示すると、ユーザーが再回答の必要性を判断しやすくなる。

### 制約1の実装仕様（確定版）

上位3位以内に同一軸のペアが両方含まれる場合：
- **スコアに差がある** → 高い方だけ残して低い方を除外。結果ページに常時表示：「陰陽ペアは排他的なため、スコアの高い方に自動で寄せています」
- **同率の場合** → 両方表示した上でワーニング：「○種と○種が同点です。どちらか一方のはずなので、【動】【顔】項目の回答を見直してみてください」
- **僅差（差が2点以内）の場合** → 寄せた上で注意書き：「○種と○種が僅差でした。【動】【顔】項目の回答を見直すと精度が上がります」（しきい値はチューニングで調整）

|---|---|---|
| 1種 | 「妄想や空想が好き」→「仮説を立てて検証するのが好き」 | 6種（非日常・ロマン）との混同を避ける |
| 2種 | 「活字や絶対的権威が書いた書物に響きやすい」→「マニュアルや前例を重視する」 | 表現を平易化、本質は維持 |
| 4種 | 「食べるのが好き・少量でも質の良いもの」を削除、食との関係を正確に記述 | 「食べるのが好き」は3種の特性。4種はネガティブな感情で食欲がなくなる（3種と正反対）。「一晩寝れば忘れてしまう」も削除（3種の切り替えの早さと混同） |
| 6種 | 「理想家で熱血漢」→「夢想家でロマンチスト」／省エネ型の損得を明示 | 「熱血漢」は7種のイメージと被る。5種との損得の違いを差別化 |
| 8種 | 「プライドが高い」→「表には出さないが対抗心が強い」 | 「プライドが高い」は7種のイメージと混同しやすく、望ましさバイアスも強い |
| 9種 | 「直感がよく働く」→「細部にまで気がつく鋭い観察眼と直感がある」 | 職人気質・一点集中との文脈的なつながりを補強 |
| 10種 | 「誰にでもおおらかで寛大・博愛主義的」→「愛情の優先順位は実はある」を追記 | taiheki.jpの考察に基づき、外からの像と実態の乖離を修正 |
