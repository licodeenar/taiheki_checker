# 体癖チェッカー

野口整体の創始者・野口晴哉が提唱した「体癖」の10種（1〜10種）について、当てはまる項目をチェックするとレーダーチャートで結果を表示するツールです。

**https://licodeenar.github.io/taiheki_checker/index.html**

## 体癖とは

人間の身体の使い方・感受性・反応パターンの「癖」を分類する概念。身体の重心軸に基づき、行動・思考・感情の傾向として現れます。

| 種 | 軸 | タイプ |
|---|---|---|
| 1種・2種 | 上下 | 頭脳型 |
| 3種・4種 | 左右 | 感情型 |
| 5種・6種 | 前後 | 損得勘定型 |
| 7種・8種 | ねじれ | 勝負型 |
| 9種・10種 | 開閉 | 愛憎型 |

## 使い方

1. 各体癖の項目を読み、当てはまるものにチェック
2. 「結果を見る」ボタンを押す
3. レーダーチャートと上位3種の解説を確認

## 注意

チェック項目はAIを用いてまとめたものです。正確な評価を求める場合は専門家にご相談ください。
